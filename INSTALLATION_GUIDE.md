# 🚀 LLM 크롤러 설치 가이드

## 📋 설치 방법 (2가지 옵션)

### **옵션 1: 간단 설치 (추천)**
```bash
# 일반 사용자 - 관리자 권한 불필요
setup.bat
```
**특징:**
- ✅ 빠르고 간단 (2-3분)
- ✅ 관리자 권한 불필요
- ✅ 핵심 기능 모두 사용 가능
- ✅ 하이브리드 크롤러로 고성능 구현
- ⚠️ aiohttp는 수동 설치 필요 (선택사항)

### **옵션 2: 완전 자동 설치**
```powershell
# PowerShell 관리자 권한에서
.\install_full.ps1
```
**특징:**
- ✅ 모든 것 자동 설치 (5-10분)
- ✅ Visual C++ 빌드 도구 자동 설치
- ✅ aiohttp 완벽 지원으로 최고 성능
- ⚠️ 관리자 권한 필요
- ⚠️ 설치 시간이 더 오래 걸림

## 🎯 어떤 방법을 선택해야 할까요?

### **→ 대부분의 사용자: `setup.bat`**
- 빠르게 시작하고 싶은 경우
- 관리자 권한을 주기 싫은 경우
- 테스트용으로 사용하는 경우

### **→ 완벽한 성능이 필요한 경우: `install_full.ps1`**
- 대용량 크롤링을 할 예정인 경우
- 최고 성능이 필요한 경우
- 개발 환경을 완벽하게 구축하고 싶은 경우

## 💡 권장 접근법

**1단계:** `setup.bat`으로 빠르게 시작
```bash
setup.bat
python quick_demo.py  # 테스트
```

**2단계:** 만족스러우면 그대로 사용, 더 나은 성능이 필요하면
```powershell
.\install_full.ps1  # 완전 설치로 업그레이드
```
